# Geomini font

![](screenshot-2026-01-27.png)

Geomini is a contemporary geometric typeface designed for clarity and versatility.

The family is authored and produced using FontBob, a browser-based font editor.

You can explore and remix this font online at:
https://fontbob.com/geomini

## Styles

- Thin
- Regular
- Medium
- SemiBold
- Bold
- ExtraBold
